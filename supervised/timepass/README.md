# 🕰️ Timepass — Scratch & Experiments

> A sandbox folder for quick, casual experiments — *"timepass"* in the Indian-English sense of pleasantly killing time.
> Think of it as the **lab bench** where ideas get tried before they graduate to a clean notebook.

---

## 📂 Contents

| File | Purpose |
|------|---------|
| `prg1.ipynb` | Free-form experimentation notebook |
| `diabetes_clean.csv` | Pima Indians Diabetes dataset (re-used) |
| `dataset_with_names.csv` | Diabetes dataset with a `Person_Name` ID column |

---

## 🎯 What's Inside

Casual try-it-out exercises with the diabetes datasets — typical scratchpad activities:

- Quick `head()` / `describe()` exploration
- Trying out different sklearn models on the fly
- Comparing how dropping vs imputing missing values affects accuracy
- Just-for-fun visualizations

---

## ▶️ Run

```bash
jupyter notebook prg1.ipynb
```

## 💡 Note

Nothing in this folder is "finished" — it's intentional. Use it as inspiration for your *own* experiments, or as a peek at the messy reality of how ML notebooks usually start.
